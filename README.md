# Node-chat-app

## Description

Simple chat application using Node.js (Express)
Based on The Complete Node.js Developer Course (3rd Edition) by Andrew Mead, available on [Udemy](https://www.udemy.com/the-complete-nodejs-developer-course-2/)

#### Dependencies

- bad-words: 3.0.3
- express: 4.17.1
- socket.io: 2.2.0
